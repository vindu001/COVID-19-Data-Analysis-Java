# COVID-19 Data Analysis Using Java

## Project Overview

A Java-based COVID-19 Data Analysis Project that processes real-world COVID-19 datasets, performs country-wise analysis, calculates recovery rates, ranks countries by total cases, and generates reports.

## Features

* COVID-19 dataset analysis using Java
* Country-wise COVID statistics
* Recovery rate calculation
* Top 10 countries ranking by total cases
* Country search functionality
* Automated report generation
* File handling and CSV processing

## Technologies Used

* Java
* File Handling
* Collections Framework
* CSV Data Processing
* Data Analysis

## Dataset

The project uses the Our World in Data COVID-19 dataset.

Note: The dataset file is not uploaded to GitHub because of its large size. Users can download it from the official source and place it inside the `data` folder.

Dataset Source:
https://ourworldindata.org/covid-cases
