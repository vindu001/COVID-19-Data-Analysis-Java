# COVID-19 Data Analysis Using Java

## Overview

A Java-based data analysis project that processes real-world COVID-19 data from the Our World in Data dataset.

## Features

* CSV File Processing
* Data Cleaning and Filtering
* India COVID Analysis
* Recovery Rate Calculation
* Top 10 Countries by Total Cases
* Country Search Functionality
* Automatic Report Generation

## Technologies Used

* Java
* File Handling
* Collections Framework
* CSV Processing
* Data Analysis

## Dataset

Our World in Data COVID-19 Dataset
